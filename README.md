# wayland dotfiles

Linux Distribution : Arch Linux

**Window Manager** : Niri<br/>
**Status Bar** : Ironbar<br/>
**Shell** : Fish<br/>
**Terminal** : WezTerm<br/>
**Launcher** : Rofi<br/>
**Editor** : Emacs<br/>
**Music** : MPD+MPC<br/>
**Browser** : Zen<br/>
**Notifications** : Mako<br/>
**Screenshot** : Niri<br/>
**PDF** : Zathura<br/>
**Image Viewer** : imv<br/>
**Video Player** : mpv<br/>
