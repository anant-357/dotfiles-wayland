# wayland dotfiles

Linux Distribution : Arch Linux

**Window Manager** : Niri<br/>
**Status Bar** : Eww<br/>
**Shell** : Nu Shell<br/>
**Terminal** : Ghossty<br/>
**Launcher** : Wofi<br/>
**Editor** : Neovim<br/>
**Music** : Spotify<br/>
**Browser** : Zen<br/>
**Notifications** : Dunst<br/>
**Screenshot** : Niri<br/>
**PDF** : Zathura<br/>
**Image Viewer** : Loupe<br/>
**Video Player** : mpv<br/>
